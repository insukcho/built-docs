Exported fieldsedit

This document describes the fields that are exported by Apm-Server. They are grouped in the following categories:

  • Alias fields
  • General APM fields
  • APM Error fields
  • APM Sourcemap fields
  • APM Span fields
  • APM Transaction fields
  • Beat fields
  • Cloud provider metadata fields
  • Docker fields
  • Host fields
  • Kubernetes fields
  • APM System Metrics fields
