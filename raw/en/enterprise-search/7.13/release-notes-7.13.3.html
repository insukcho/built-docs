7.13.3 release notesedit

Upgrading to Enterprise Search 7.13.3? See Upgrading & migrating.

  • Enterprise Search
  • Workplace Search

Enterprise Searchedit

Bug fixesedit

  • Disables fuzziness for long queries when the query contains multi-byte characters. Prevents long queries in some languages from causing Elasticsearch errors and poor performance.

Workplace Searchedit

Bug fixesedit

  • Fixes issue where a user that has been deleted in a third-party service and mapped as an external identity would cause the permissions job to fail.
