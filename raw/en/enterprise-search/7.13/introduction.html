Introductionedit

Elastic Enterprise Search packages Elastic App Search and Elastic Workplace Search into a single solution.

Topics relevant to both products, such as installation and configuration are covered here.

Topics specific to each product are covered elsewhere:

  • Elastic App Search documentation
  • Elastic Workplace Search documentation

For versions 7.9.0 through 7.13.0, Enterprise Search is available within Kibana as an early preview release. That release is not intended for production use. Enterprise Search in Kibana is available in beta since 7.14.0 and will be generally available in Enterprise Search 7.16.0.
