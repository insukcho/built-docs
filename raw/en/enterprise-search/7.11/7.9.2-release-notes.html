7.9.2 Release notesedit

Upgrading to Enterprise Search 7.9.2? See Upgrading & migrating.

Enterprise Searchedit

Bug fixesedit

  • Fixes users not being redirected after logging in with SAML.
  • Fixes Docker image missing libfontconfi1 font library.
  • Fixes Enterprise Search not waiting long enough when migrating very large deployments from legacy App Search.

Known issuesedit

  • Elastic Cloud on Kubernetes (ECK) deployments cannot launch the Enterprise Search UI from Kibana.

    To work around this limitation, launch the Enterprise Search UI directly, not through Kibana.
