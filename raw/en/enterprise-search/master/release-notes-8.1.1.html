8.1.1 release notesedit

Upgrading to Enterprise Search 8.1.1? See Upgrading & migrating.

Bug fixesedit

  • The Enterprise Search panel is now hidden in the Cluster Overview when no nodes are present.

Known issuesedit

  • Jira Cloud and Confluence Cloud content sources cannot be newly connected or reauthorized in 8.1.1. This is due to backwards incompatible API changes in these Atlassian products that were subsequently rolled back.
  • App Search users with dev and editor roles do not have access to the web crawler.
