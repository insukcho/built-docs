Web crawleredit

The Elastic Web Crawler programmatically discovers, extracts, and indexes content from public-facing websites and knowledge bases. Use the web crawler to transform webpage content into searchable content, and keep this content in sync with webpage changes over time. The web crawler is a powerful and flexible tool for ingesting content into your Elastic deployment.

The web crawler is available for use with any Elastic deployment. Unlike its predecessor, the App Search web crawler, the web crawler is a native Elasticsearch solution— it reads and writes directly to Elasticsearch indices. Use the full power and flexibility of Elasticsearch to search and analyze your crawled data.
