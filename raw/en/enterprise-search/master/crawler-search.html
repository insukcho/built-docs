Learn how to search your crawled documents, using a host of Elasticsearch options, and App Search.

You have a wide range of native Elasticsearch search options:including the REST API, the Kibana Dev Tools, programming language clients, etc. You can also use the Search UI with the Elasticsearch connector. If you index the documents into an App Search engine, using BYOEI, you can use the App Search search functionalities to search over your documents.
