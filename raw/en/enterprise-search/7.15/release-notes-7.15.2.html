7.15.2 release notesedit

Upgrading to Enterprise Search 7.15.2? See Upgrading & migrating.

Enterprise Searchedit

Bug fixesedit

  • Fixes starting Enterprise Search when security is disabled.

App Searchedit

Bug fixesedit

  • Fixes issue related to meta engine and boosts that could cause the engine overview UI to become inaccessible.
  • Fixes an issue where the web crawler was using incorrect units for HTTP timeout configuration.

Known issuesedit

  • App Search users with dev and editor roles do not have access to the web crawler.
