Workplace Searchedit

Bug fixesedit

  • Fixes error introduced in 7.12.0 that affects custom sources migrated from earlier versions of Enterprise Search.
  • Fixes Confluence Server connections looping when a large number of Pages are created in a single Space within a small timeframe.
  • Fixes a cause of Access token is invalid! errors for Slack connections. The bug impacted Slack OAuth apps created after 2021-02-24.
