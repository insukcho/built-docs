Azure region Central India (Pune) now availableedit

Elasticsearch Service is now available in the Azure Central India (Pune) region. You can select the new region when you create new deployments.

Like other recently added regions, Pune supports only HTTPS access for additional security. To learn more about what access different regions support, see Elasticsearch Service regions.
