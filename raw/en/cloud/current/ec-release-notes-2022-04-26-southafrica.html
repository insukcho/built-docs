Azure region South Africa North (Johannesburg) now availableedit

Elasticsearch Service is now available in the Azure South Africa North (Johannesburg) region. You can select the new region when you create new deployments.

Like other recently added regions, Johannesburg supports only HTTPS access for additional security. To learn more about what access different regions support, see Elasticsearch Service regions.
