IpFilterRulesetsedit

The container for a set of IP filter rulesets.

Propertiesedit

rulesets (array[IpFilterRuleset], required)
List of rules

Exampleedit

{
   "rulesets" : [
      {
         "associations" : [
            {
               "entity_type" : "string",
               "id" : "string"
            }
         ],
         "description" : "string",
         "id" : "string",
         "name" : "string",
         "rules" : [
            {
               "description" : "string",
               "id" : "string",
               "source" : "string"
            }
         ],
         "total_associations" : 0
      }
   ]
}
