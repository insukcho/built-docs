Get instance typesedit

Retrieves a list of all instance types.

Requestedit

GET /api/v1/stack/instance-types

Responsesedit

200

(array[InstanceTypeResource]) List of node types

To perform this operation, you must be authenticated by means of one of the following methods: apiKey, basicAuth.

Request exampleedit

curl -XGET {{hostname}}/api/v1/stack/instance-types \
-H "Authorization: ApiKey $ECE_API_KEY"
