ProxySummaryedit

Summarized information about a proxy.

Propertiesedit

healthy (boolean, required)
Whether this proxy is healthy
proxy_id (string, required)
The proxy identifier

Exampleedit

{
   "healthy" : true,
   "proxy_id" : "string"
}
