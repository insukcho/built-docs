ConfigStoreOptionListedit

A list of Config Store values.

Propertiesedit

values (array[ConfigStoreOption], required)
List of Config Store Options

Exampleedit

{
   "values" : [
      {
         "changed" : true,
         "name" : "string",
         "value" : "string"
      }
   ]
}
