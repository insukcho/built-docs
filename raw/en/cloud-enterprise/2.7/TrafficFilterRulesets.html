TrafficFilterRulesetsedit

The container for a set of traffic filter rulesets.

Propertiesedit

rulesets (array[TrafficFilterRulesetInfo], required)
List of traffic filter rules

Exampleedit

{
   "rulesets" : [
      {
         "associations" : [
            {
               "entity_type" : "string",
               "id" : "string"
            }
         ],
         "description" : "string",
         "id" : "string",
         "include_by_default" : true,
         "name" : "string",
         "region" : "string",
         "rules" : [
            {
               "description" : "string",
               "id" : "string",
               "source" : "string"
            }
         ],
         "total_associations" : 0,
         "type" : "string"
      }
   ]
}
