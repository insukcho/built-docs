IpFilterRuleedit

The container for an IP filter rule.

Propertiesedit

description (string)
Description of the rule
id (string)
The rule ID
source (string, required)
IPV4, IPV6 address or CIDR mask to define single or range of IPs

Exampleedit

{
   "description" : "string",
   "id" : "string",
   "source" : "string"
}
