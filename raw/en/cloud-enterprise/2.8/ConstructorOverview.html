ConstructorOverviewedit

The overview information for the installed constructors.

Propertiesedit

constructors (array[ConstructorInfo], required)

Exampleedit

{
   "constructors" : [
      {
         "constructor_id" : "string",
         "status" : {
            "connected" : true,
            "maintenance_mode" : true
         }
      }
   ]
}
