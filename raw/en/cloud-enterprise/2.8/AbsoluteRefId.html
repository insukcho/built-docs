AbsoluteRefIdedit

A reference to a specific resource of a deployment

Propertiesedit

deployment_id (string, required)
The deployment id
ref_id (string, required)
The reference id of the resource in the given deployment

Exampleedit

{
   "deployment_id" : "string",
   "ref_id" : "string"
}
