TrafficFilterRuleedit

The container for an IP filter rule.

Propertiesedit

description (string)
Description of the rule
id (string)
The rule ID
source (string, required)
Allowed traffic filter source: IP address, CIDR mask, or VPC endpoint ID

Exampleedit

{
   "description" : "string",
   "id" : "string",
   "source" : "string"
}
