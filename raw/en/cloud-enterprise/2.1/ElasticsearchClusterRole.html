ElasticsearchClusterRoleedit

A list of users together with that user's roles

Propertiesedit

roles (array[string], required)
The list of roles for this user
username (string, required)
The username

Exampleedit

{
   "roles" : [
      "string"
   ],
   "username" : "string"
}
