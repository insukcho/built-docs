DeleteUsersApiKeysRequestedit

The request payload that deletes the API keys.

Propertiesedit

user_api_keys (array[UserApiKey], required)
The list of API key IDs.

Exampleedit

{
   "user_api_keys" : [
      {
         "api_key_id" : "string",
         "user_id" : "string"
      }
   ]
}
