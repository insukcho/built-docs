App Search templateedit

This template provides the capabilities of App Search, built on top of the Elastic Cloud Enterprise, to deliver a rich search experience within your applications. App Search features include a set of RESTful APIs to ingest data, real-time analytics, relevance tuning, query synonym support, and an intuitive dashboard.

To enable App Search on Elastic Cloud Enterprise, see Enable App Search.

For full App Search documentation, see the App Search Reference.

Included in this template:

  • Elasticsearch data nodes: Start at 1 GB memory x 1 availability zone. Use the data.default instance configuration.
  • App Search nodes: Start at 2 GB memory x 1 availability zone. Use the appsearch instance configuration.
  • Kibana: Starts at 1 GB memory x 1 availability zone. Uses the kibana instance configuration.
