Searching with Workplace Searchedit

Workplace Search provides a modern web-based search experience for all types of individuals in the workforce, and a powerful programmatic search interface for custom implementations.

This section assumes you have already invited users to the Workplace Search experience, and connected a variety of content sources to the platform. If you aren’t up and running yet, read the Getting started guide for guidance through the initial setup steps.

Get searching with the following guides:

  1. The searcher’s guide: Teach your team mates The Ways of Search with Workplace Search.
  2. Searching from your web browser’s address bar: Save time; initiate your searches directly from your browser’s address bar.
  3. Custom search experiences: Learn more about the Search API and how to embed search within any internal application.
