Min Score Usageedit

Fluent DSL exampleedit

s => s
.MinScore(0.5)
.Query(q => q
    .Term(p => p.Name, "elasticsearch")
)

Object Initializer syntax exampleedit

new SearchRequest<Project>
{
    MinScore = 0.5,
    Query = new TermQuery
    {
        Field = "name",
        Value = "elasticsearch"
    }
}

Example json output.

{
  "min_score": 0.5,
  "query": {
    "term": {
      "name": {
        "value": "elasticsearch"
      }
    }
  }
}
